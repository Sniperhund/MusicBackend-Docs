## GET /user/tracks/contains

Check if a track is saved by the user

:::info

You have to attach a access token in the Authorization in the header

:::

### Request

#### Query Parameters

-   `id` (mongodb id) - The id of the track.

### Response

<Tabs>
  	<TabItem value="200" label="200" default>
        User's saved tracks found.

        ```js
        {
            "status": "ok",
            "contains": "true",
        }
        ```
    </TabItem>
    <TabItem value="400" label="400">
        No saved tracks found.

        ```js
        {
            "status": "error",
            "message": "No saved tracks found"
        }
        ```
    </TabItem>

</Tabs>
