## GET /user/tracks

Get user's saved tracks

:::info

You have to attach a access token in the Authorization in the header

:::

### Request

#### Query Parameters

-   `limit` (integer) - Limit the number of tracks to be returned. Default is 20.
-   `offset` (integer) - Offset the tracks to be returned. Default is 0.
