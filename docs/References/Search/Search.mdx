## POST /auth/register

Register a user

### Request

#### Query Parameters

-   `q` (string, required): Search query
-   `type` (string enum ["track", "album", "artist", "default"]):
    -   The type of item to search for.
    -   If not provided, the default is "default".
    -   If "default", the search will include all types of items. Where is 1/6 is artist, 1/6 is album, 4/6 is track.
-   `limit` (integer): The number of items to return. Default is 12.

:::info

You have to attach a access token in the Authorization in the header

:::
