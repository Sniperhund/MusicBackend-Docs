## GET /albums/:id/tracks

Get all the tracks of an album.

:::info

You have to attach a access token in the Authorization in the header

:::

### Request

#### Query Parameters

-   `limit` (integer) - The number of tracks to return.
-   `offset` (integer) - The number of tracks to skip before starting to return tracks.
