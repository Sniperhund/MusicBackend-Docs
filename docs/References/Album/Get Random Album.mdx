## GET /albums/genre/:id/random

Get random albums by genre.

:::info

You have to attach an access token in the Authorization header.

:::

### Request

#### Path Parameters

-   `id` (string, required) - The ID of the genre to filter albums.

#### Query Parameters

-   `limit` (integer, optional) - The number of albums to retrieve. Default is 10.
